# AI-Powered Learning Assistant (Subsite for SMS+LMS)

A full-stack, AI-powered Learning Assistant app designed as a **subsite of the main SMS+LMS system**.

## Core Functionalities Implemented

1. **User Authentication** – Secure login and signup with JWT.
2. **PDF Upload & Management** – Upload, store, and manage study documents with file-size tracking.
3. **Embedded PDF Viewer** – Read documents directly inside the app.
4. **AI-Powered Chat** – Ask questions about uploaded documents and get context-aware responses via Google Gemini.
5. **AI Document Summary** – Generate concise one-click summaries of entire documents.
6. **AI Concept Explainer** – Receive detailed explanations of specific topics from document content.
7. **Auto-Generated Flashcards** – Create flashcard sets from document content with flip animation.
8. **AI Quiz Generator** – Generate configurable multiple-choice quizzes.
9. **Quiz Results & Analytics** – Review scores, correct answers, and explanations.
10. **Progress Tracking Dashboard** – Track total documents, flashcards, quizzes, and recent activity.
11. **Favorites System** – Mark important flashcards for quick review.
12. **Responsive UI** – Modern, mobile-friendly interface built with Tailwind CSS.

## Platform Positioning

> This Learning Assistant is intended to run as a **subsite/module within the broader SMS+LMS ecosystem**, not as a standalone primary platform.
